🛍️ Retail Trend Analytics

An interactive SQL-driven dashboard analyzing supermarket sales performance through revenue trends, customer behavior, product line analysis, and branch-level insights.

📈 Key Features

KPI overview: Total Revenue, Transactions, Units Sold, Avg Rating, Gross Income, Avg Transaction Value

Revenue by Product Line analysis

Revenue by Branch breakdown

Quantity Sold by Product Line insights

Average Rating by Product Line comparison

Payment Method Distribution analysis

Revenue by Customer Type and Gender segmentation

🗂️ Dataset

Retail dataset sourced from Kaggle.

🛠️ Tools

SQL, SQLite Viewer, HTML, JSON, Excel

🎯 Skills

SQL Analysis • Data Visualization • Dashboard Design • Retail Analytics • Business Intelligence
